# VSCodeSnap

📸 在 VS Code 中为您的代码创建精美截图！

## 关于此项目

此项目是 [CodeSnap](https://github.com/luisllamasbinaburo/VSCodeSnap) 的一个分支项目，原作者是 luisllamas。感谢原作者的贡献。

## 主要功能

- 创建代码的精美截图
- 支持多种背景和样式选项
- 支持复制到剪贴板或保存为文件
- 可自定义窗口标题和控制按钮样式

## 使用方法

1. 选择要截图的代码
2. 右键点击选择 "VSCodeSnap 📸" 或使用命令面板
3. 调整预览窗口
4. 点击 "Copy" 复制到剪贴板或 "Save" 保存为文件
